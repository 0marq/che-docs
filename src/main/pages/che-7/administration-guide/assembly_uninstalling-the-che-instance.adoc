= Uninstalling the Che instance

:context: uninstalling-the-che-instance

The following section describes uninstallation procedures for Eclipse Che installed on the OpenShift Container Platform. The uninstallation process will lead to the complete removal of Che-related user data. The appropriate uninstallation method depends on what method was used to install the Che instance.

* If Che was installed using OperatorHub, see xref:uninstalling-the-che-instance-after-operatorhub-installation_uninstalling-the-che-instance[]
* If Che was installed using chectl, see xref:uninstalling-the-che-instance-using-chectl-after-chectl-installation_uninstalling-the-che-instance[]
